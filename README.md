# Healthcare Enrollment Analytics Dashboard

## Overview

This project is an end-to-end Business Intelligence solution built with
**Power BI** and **Microsoft SQL Server** to analyze healthcare
enrollment operations. The dashboard provides insights into enrollment
performance, outreach activities, referrals, renewals, geographic
distribution, and member demographics through interactive visualizations
and executive-level KPIs.

The project includes SQL data modeling, reporting views, DAX
calculations, and Power BI dashboards designed to support operational
and strategic decision-making in a healthcare environment.

------------------------------------------------------------------------

## Objectives

-   Monitor healthcare enrollment performance.
-   Track outreach and referral activities.
-   Analyze renewal trends.
-   Identify geographic enrollment patterns.
-   Evaluate member demographics.
-   Deliver actionable business insights through interactive dashboards.
